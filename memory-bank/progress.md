## 技术栈变更说明

| 原计划 | 实际实现 | 原因 |
|--------|----------|------|
| Vue 3 | React 19 | React生态更成熟，组件化开发更灵活 |
| Pinia | React Context | 项目规模适中，Context足够使用 |
| IndexedDB Promised | Dexie.js | Dexie提供更简洁的API |
